# Security Policy

## Supported Versions

Security updates are provided for the current production version of MI LinerLens.

## Reporting a Vulnerability

Please report suspected security vulnerabilities by email:

**dpo@milkrite-interpuls.com**

Include as much detail as possible, such as:

- Affected URL, endpoint, or component
- Steps to reproduce
- Potential impact
- Screenshots, logs, or proof of concept details, if available

Please do not publicly disclose the vulnerability until it has been reviewed and addressed.

We will review reports and follow up as soon as possible.
